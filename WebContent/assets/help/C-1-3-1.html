共享文档

在会议中，主持人、主讲人以及具有文档共享权限的与会者，可以共享Word、Excel、PDF、PPT等所有支持打印的文档。

不同的角色，共享文档的操作权限不同，如表所示。

共享文档权限

角色

权限

主持人

  • 查看共享的文档内容。
  • 打开/关闭、保存、翻页。
  • 使用自带的标注工具对共享的文档进行文字、图形的标注。
  • 同步会议内容。
  • 以各种比例、实际大小、适合页面、适合宽度以及全屏的方式显示共享的文档。
  • 设置与会者共享文档的权限。

主讲人

  • 查看共享的文档内容。
  • 打开/关闭、保存、翻页。
  • 关闭自己或其他与会者共享的文档。
  • 使用自带的标注工具对共享的文档进行文字、图形的标注。
  • 同步会议内容。
  • 以各种比例、实际大小、适合页面、适合宽度以及全屏的方式显示共享的文档。
  • 切换数据共享的方式。

参与者

  • 查看共享的文档内容。
  • 同步会议内容。
  • 以各种比例、实际大小、适合页面、适合宽度以及全屏的方式显示共享的文档。

➢ 开始共享文档

您可以通过以下两种方式开始共享文档：

  • 在工具栏中，单击，选择“共享文档”，从本地选择需要共享的文档。
  • 在会议信息界面的快捷入口中，选择“共享文档”，从本地选择需要共享的文档。
说明：
  • 共享文档时，支持共享小于等于50MB的文档且支持显示的页数为200页。如果共享文档的总页数超过200页，则显示前200页内容。
  • 共享动态的PPT文件时，不支持显示动态效果。
  • 共享文档时，您只能关闭自己共享的文档。

➢ 了解共享文档中的工具条

开始共享后，您可以根据自己的习惯，选择全屏模式或者非全屏模式。不同的查看模式，系统默认提供的工具条也有所不同。

  • 非全屏模式下，共享文档可以操作的工具条，如图所示，功能说明如表所示。

➢ 在文档中标注

主持人、主讲人和赋予标注权限的与会者都可以在文档中进行各种标注。

  • 非全屏模式下可以直接使用标注工具。
  • 全屏模式下单击激活标注工具，如图所示。

标注工具

标注工具从左到右依次为：“选择”、“笔”、“线框”、“文本标注”、“勾与叉”、“擦除标注”、“设置显示比例”、“退出标注”。

说明：

主讲人可以删除自己的标注，也可以删除主持人和参与者的标注；主持人和参与者只能删除自己的标注。

➢ 与其他与会者发送消息

1. 单击，系统显示与会者列表面板。

2. 单击，系统显示即时消息面板。

3. 在与会者列表中，右键单击与会者，选择“发送即时消息”。

4. 输入消息内容。

5. 按“Enter”键，发送消息。
